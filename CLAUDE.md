# 八十！八十！

一款使用 Phaser 3 开发的益智解谜游戏，支持发布到微信、抖音、支付宝小游戏平台。

## 核心玩法
1. 在积木块拼成的墙上抡起大锤砸，砸掉一块块积木，直到整面墙重心失去支撑垮塌为止。
2. 每局有总预算（比如1000元），砸一下80，如果预算花光了墙没倒塌则闯关失败。如果过关剩余预算装入私房钱袋。
3. 每3关可以进商店买道具，用私房钱袋里的钱。道具有力量加持的锤子（砸1下抵1.5下）、重心分析镜（标注重心点）、蛀虫（让积木块的硬度降级）。
4. 按用户所在地区分组、把私房钱求和，展示区域私房钱排行榜；按过关数展示个人排行榜。

## 游戏控制

### 鼠标点击
点到的积木块显示砸墙效果，然后消失，计算重心是否有支撑。

### 触摸控制
在移动端用单点触控代替鼠标点击。

## 游戏声音
有背景音和砸墙音效

## 游戏画面
墙体积木块要有贴图或纹理，不能纯色太单调。不同的积木可以有不同的贴图代表抗砸次数，比如泥土块砸1次、瓷砖块砸2次、大理石3次、金属的5次。

## 多端匹配
至少要能在PC网页和微信小游戏中运行，后续要适配抖音、快手等小游戏平台。

## 产品里程碑
1. 分3个里程碑
2. milestone1 纯PC端网页游戏，实现核心玩法1和2
3. milestone2 增加后端、适配微信小游戏，实现核心玩法3和4
4. milestone3 前端适配其他指定游戏平台，后端跨平台统计数据

## 开发建议
1. 前端使用Phaser 3框架，酌情引入typescript
2. 后端使用Golang或Rust，数据库用PostgreSQL
3. 按软件工程执行，保证开发质量和版本控制
4. 需要外部资源时立即停下来求助
